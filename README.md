# Miniguia de Estudos com NotebookLM: Fundamentos de Cibersegurança

## 📌 Contexto
Este projeto foi desenvolvido como parte do desafio da DIO com o objetivo de explorar o uso do NotebookLM como ferramenta de aprendizagem ativa.

O tema escolhido foi **Fundamentos de Cibersegurança**, uma área essencial para profissionais de tecnologia que desejam compreender ameaças, vulnerabilidades e mecanismos de proteção digital.

## 🎯 Objetivos de Aprendizagem
- Compreender os pilares da segurança da informação.
- Identificar os principais tipos de ameaças cibernéticas.
- Entender conceitos como autenticação, criptografia e controle de acesso.
- Desenvolver uma base sólida para estudos avançados em segurança.

## 🤖 Engenharia de Prompts
### Prompt 1
"Explique os três pilares da segurança da informação com exemplos práticos."

**Resultado:** Resposta clara e objetiva sobre confidencialidade, integridade e disponibilidade.

### Prompt 2
"Compare phishing, ransomware e malware em uma tabela."

**Resultado:** Excelente para visualização comparativa.

### Prompt 3

**Resultado:** Estrutura progressiva e prática.

## ⚠️ Desafios Encontrados
- Alguns prompts geraram respostas muito genéricas.
- Foi necessário refinar perguntas para obter exemplos mais técnicos.
- A qualidade da resposta melhorou ao especificar contexto e nível de profundidade.

## 📖 Miniguia de Estudos

### 1. Pilares da Segurança da Informação
- **Confidencialidade:** proteger informações contra acesso não autorizado.
- **Integridade:** garantir que os dados não sejam alterados indevidamente.
- **Disponibilidade:** assegurar acesso quando necessário.

### 2. Principais Ameaças
- Phishing
- Ransomware
- Malware
- Engenharia Social
- Ataques DDoS

### 3. Controles de Segurança
- Autenticação multifator (MFA)
- Criptografia
- Firewalls
- Antivírus
- Backup

## 📘 Glossário
- **Malware:** software malicioso.
- **Phishing:** tentativa de roubo de dados por fraude.
- **Firewall:** sistema de filtragem de tráfego.
- **Criptografia:** proteção de dados por codificação.
- **MFA:** autenticação em múltiplas etapas.

## 🚀 Conclusão
O NotebookLM demonstrou ser uma ferramenta poderosa para aprendizagem ativa, permitindo organizar fontes, gerar insights e aprofundar conhecimentos de forma estruturada.
