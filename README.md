# miniguia-ciberseguranca-notebooklm
Projeto desenvolvido com o NotebookLM para estudar Fundamentos de Cibersegurança. O trabalho inclui curadoria de fontes, engenharia de prompts, documentação de testes e criação de um miniguia estruturado de estudos, demonstrando o uso da IA como ferramenta de aprendizagem ativa.
